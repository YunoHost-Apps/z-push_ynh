## Przegląd

Z-Push is an Exchange ActiveSync fronted written in PHP which lets you synchronize emails (IMAP/SMTP backend) and calendar/contacts (cardDAV and caldDAV backend)


**Dostarczona wersja:** 2.7.5~ynh1
